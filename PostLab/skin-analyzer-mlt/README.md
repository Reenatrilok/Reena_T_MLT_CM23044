# AI Skin & Face Wellness Analyzer

## Description
This project uses computer vision to analyze facial features and provide wellness suggestions such as sleep improvement, hydration, and skincare.

## Features
- Face Detection using OpenCV & MediaPipe
- Dark Circle Detection
- Skin Brightness Analysis
- Acne Detection
- Real-time Suggestions

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run
pip install -r requirements.txt
python main.py
